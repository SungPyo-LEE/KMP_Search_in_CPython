# KMP_Search_in_CPython
## C로 씌여진 문자열 탐색, 파이썬으로 임베딩 하자

자연어 처리 된 문자를 문서에서 빠르게 찾기 위해 KMP 알고리즘을 적용. Python에서 탐색은 느려서 C로 임베딩 된 함수를 이용 해보자
